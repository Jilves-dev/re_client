Space realization app experimental

MERN Stack (MongoDB, Express, React, Node.js) with a fully responsive TailwindCSS UI. Includes Google Maps / Places integration, a comprehensive admin dashboard, multi-image uploads via AWS Storage, and post analytics (views, likes, messages). Admin panel features message replies through AWS SES. Unfortunately slightly slow download due to free versions of MongoDB/Vercel.
